Списки

Здесь рассматриваются списки, в которых каждый элемент начинается с новой строки. Как правило, элементы списка предваряются обобщающим текстом.

Во всех случаях первый элемент перечня ставится ровно на следующую строку за обобщающим текстом без увеличения расстояния между строками. При этом между элементами списка оставляется дополнительное расстояние для визуального выделения отдельных элементов.

Арабскими цифрами с точкой обозгначают элементы, если каждый элемент является отдельным предложением, т. е. начинается с прописной буквы и оканчивается точкой.

Нумерованный список:
  1. 25 панелей (дисплеев) c диагональю 40–55".
  2. Видеостена типового размера 2x2 или 3x3 из панелей 40–55" (размер на усмотрение поставщика).
  3. Светодиодная панель для установки на улице (размер на усмотрение поставщика).
    1. Крупные кадры «Голова и плечи».
    2. Чистота отображения.
    3. Визуальный контакт.
